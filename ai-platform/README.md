# 🚀 AI Platform

A multi-functional AI platform built using the Gemini API.
This project integrates multiple AI capabilities into a single application.

---

## 🧠 Features

### 🤖 Chat Agent

* Conversational AI with memory
* Context-aware responses

### 📄 Document Analyzer

* PDF and TXT support
* Multi-file upload
* Summary, key points, and critique

### 🖼️ Image Analyzer

* Image understanding
* Multi-image support
* Contextual descriptions

---

## 🛠️ Technologies Used

* Python
* Gemini API
* Streamlit
* PyPDF2
* Pillow

---

## ⚙️ Setup

```bash
git clone https://github.com/seu-usuario/ai-platform.git
cd ai-platform
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=your_api_key_here
```

---

## ▶️ Run the Application

```bash
uv run streamlit run app.py
```
