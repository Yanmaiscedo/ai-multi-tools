# 📄 PDF Reader

A tool that extracts and analyzes content from PDF files using AI.

---

## 🚀 Features

* Text extraction from PDF files
* Content summarization
* Question answering based on document

---

## ⚙️ How to Run

```bash
uv init .
uv add streamlit openai PyPDF2 python-dotenv 
uv run streamlit run main.py 
```

---

## 🧠 Use Case

This module shows how AI can be used to process and understand documents, enabling automation of reading and analysis tasks.
