# 🤖 AI Agent

An intelligent AI agent built using the OpenAI API, capable of understanding user input and performing tasks through prompt-based reasoning.

---

## 🚀 Features

* Conversational interaction
* Task execution based on prompts
* Context-aware responses

---

## ⚙️ How to Run

```bash
uv init .
uv add langgraph langchain python-dotenv langchain-openai
uv run main.py
```

---

## 🧠 Use Case

This module demonstrates how to build a basic AI agent that can simulate reasoning and assist users in real-world tasks.
