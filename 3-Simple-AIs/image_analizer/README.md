# 🖼️ Image Analyzer

An AI-powered tool for analyzing and understanding images.

---

## 🚀 Features

* Image description generation
* Visual content analysis
* Multimodal AI interaction

---

## ⚙️ How to Run

```bash
uv init .
uv add streamlit tensorflow opencv-python
uv run streamlit run main.py 
```

---

## 🧠 Use Case

This module demonstrates how to integrate computer vision with AI models to extract insights from images.
